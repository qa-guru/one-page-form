# One Page Form Demos

Small static HTML/CSS/JavaScript demos for practicing form interactions and basic UI behavior.

## Project Structure

- `text-box.html` - Simpler text box form demo that renders submitted values on the page.
- `automation-practice-form.html` - Full "Automation Practice Form" demo with validation, custom inputs, and result modal.

## Features

### `automation-practice-form.html`

- Client-side validation before submit
- Custom subjects autocomplete with removable chips
- Date of birth handling and normalized value output
- Picture file name capture
- State/City selection logic
- Submission preview in a modal dialog

### `text-box.html`

- Basic text and textarea fields
- Submit handler with live result rendering
- Responsive header/navigation behavior with mobile burger menu

- This repository is static front-end practice code.
- Data is not persisted; submissions are rendered in-memory in the current page session.
